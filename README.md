# Wanderlust

Wanderlust is a travel listing web application built as a project to practice full-stack web development concepts.

The project is inspired by travel and stay-listing platforms where users can explore different places, view listing details, and understand how real-world listing-based websites are structured.

## 🚀 Project Overview

This project helped me learn how to build a complete web application with proper routing, pages, UI structure, backend handling, and database-connected features.

## ✨ Features

- Travel listing pages
- Listing details page
- Clean and responsive UI
- User-friendly navigation
- Full-stack project structure
- Backend route handling
- Database-connected listings
- Practical project-based learning

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
- EJS / React.js
- Git & GitHub

## 📚 What I Learned

While building this project, I learned:

- How to structure a full-stack web application
- How routing works in backend projects
- How to connect a project with a database

- How to design user-friendly pages
- How to organize project files properly
- How to debug and improve a real project

## 🎯 Purpose

The main purpose of this project is to improve my full-stack development skills and understand how travel/listing-based platforms work.
