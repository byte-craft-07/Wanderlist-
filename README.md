# Wanderlist

A travel listing web app built with Express, EJS, MongoDB, Passport, and Mongoose.

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Copy `.env.example` to `.env` and update values if needed.

3. Start MongoDB locally.

4. Start the app:

   ```bash
   npm start
   ```

The app runs on `http://localhost:8080`.
